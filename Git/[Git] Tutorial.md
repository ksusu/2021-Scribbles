## Git Bash Command (Similar to Linux)
-   화면 초기화 : **Ctrl + L**
-   한 행의 처음과 끝 : **Ctrl + A**, **Ctrl + E**
-   목록 보기 : **ls** 또는 **dir**
-   파일의 내용 보기 : **cat**
-   특정 문자를 검색 : **grep**
-   디렉터리로 이동 : **cd**
-   디렉터리 생성 : **mkdir**
-   파일 삭제 : **rm**
-   파일 생성 : **touch**
